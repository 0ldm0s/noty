Noty Options

Available options listed below. 
	
$.noty.defaultOptions = {
	layout : 'top', // (top, topLeft, topCenter, topRight, bottom, center, bottomLeft, bottomRight)
	theme : 'noty_theme_default',	// theme name (accessable with CSS)
	animateOpen : {height: 'toggle'}, // opening animation
	animateClose : {height: 'toggle'}, // closing animation
	easing : 'swing', // easing
	text : '', // notification text
	type : 'alert', // noty type (alert, success, error)
	speed : 500, // opening & closing animation speed
	timeout : 5000, // delay for closing event. Set false for sticky notifications
	closeButton : false, // enables the close button when set to true
	closeOnSelfClick : true, // close the noty on self click when set to true
	closeOnSelfHover : false, // close the noty on self mouseover when set to true
	force : false, // adds notification to the beginning of queue when set to true
	onShow : false, // callback for on show
	onClose : false, // callback for on close
	buttons : false, // an array of buttons
	modal : false, // adds modal layer when set to true
	template: '<div class="noty_message"><span class="noty_text"></span><div class="noty_close"></div></div>',
	cssPrefix: 'noty_', // this variable will be a type and layout prefix.
	custom: {
		container: null // $('.custom_container')
	}
};
Name Default Description
text string Notification message
layout top, topCenter, bottom, center, topLeft, topRight, bottomLeft, bottomRight Layout of notification. (Customizable with CSS)
theme noty_theme_default Theme of notifications. (Customizable with CSS)
template <div class="noty_message"><span class="noty_text"></span><div class="noty_close"></div></div> Template must have at least noty_message, noty_text and noty_close class.
cssPrefix noty_ This variable will be a type and layout prefix.
type alert, error, success Style of the notification. (Customizable with CSS)
modal false Adds modal layer when set to true.
speed 500 (ms) Speed of open and close animations.
timeout 5000 (ms), false Delay for closing event. Set false for sticky notification.
animateOpen {height: 'toggle'} A map of CSS properties.
animateClose {height: 'toggle'} A map of CSS properties.
easing swing Easing effect.
closeButton false Enables the close button when set to true.
closeOnSelfClick true Close the noty on self click when set to true.
closeOnSelfHover false Close the noty on self mouseover when set to true.
force false Adds notification to the beginning of queue when true.
buttons false, array An array of buttons.
onShow false, function onShow Callback (Details are in Callbacks section)
onClose false, function onClose Callback (Details are in Callbacks section)
custom.container null, element Custom container for noty
