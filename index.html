Welcome

noty is a jQuery plugin that makes it easy to create alert, success, error and confirmation messages as an alternative the standard alert dialog. Each notification is added to a queue. (Except growl like notifications)

The notifications can be positioned at the top, topCenter (like GMail), bottom, center, top left, top right, bottom left or bottom right.

There are lots of other options in the API to customise the text, animation, speed, buttons and much more.

It also has various callbacks for the buttons, opening closing the notifications and queue control.

Installation

Include jQuery latest release in your header. The Google AJAX Libraries API can be used for this but you could also host the library yourself.

Download noty from the Github and upload the files to your server.

Include jquery.noty.js and its dependancies below jQuery.
Also add jquery.noty.css and noty_theme_default.css to your header. Or some other theme.

noty is compatible with jQuery 1.6+ for now. But if you are using older version then 1.6 you can add promise.js to your header. (Thanks for Maksim Pecherskiy)

Your code should then be similar to this: 
							
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.noty.js"></script>

<link rel="stylesheet" type="text/css" href="css/jquery.noty.css"/>
<link rel="stylesheet" type="text/css" href="css/noty_theme_default.css"/>

Creating a noty

We wrote a helper for creating noty easily. So you can use noty(properties); Just like this; 
							
// ex1 - alert
$('.ex1.alert').click(function() {
	noty({text: 'noty - a jquery notification library!'});
});

Noty Options

Available options listed below. 
							
$.noty.defaultOptions = {
	layout : "top", // (top, topLeft, topCenter, topRight, bottom, center, bottomLeft, bottomRight)
	theme : "noty_theme_default",	// theme name (accessable with CSS)
	animateOpen : {height: 'toggle'}, // opening animation
	animateClose : {height: 'toggle'}, // closing animation
	easing : 'swing', // easing
	text : "null", // notification text
	textAlign : "center", // notification text-align
	type : "alert", // noty type (alert, success, error)
	speed : 500, // opening & closing animation speed
	timeout : 5000, // delay for closing event. Set false for sticky notifications
	closable : false, // enables the close button when set to true
	closeOnSelfClick : true, // close the noty on self click when set to true
	closeOnSelfHover : false, // close the noty on self mouseover when set to true
	force : false, // adds notification to the beginning of queue when set to true
	onShow : false, // callback for on show
	onClose : false, // callback for on close
	buttons : false, // an array of buttons
	modal : false // adds modal layer when set to true
};
Name Default Description
text string Notification message
textAlign center Alignment of message
layout top, topCenter, bottom, center, topLeft, topRight, bottomLeft, bottomRight Layout of notification. (Customizable with CSS)
theme noty_theme_default Theme of notifications. (Customizable with CSS)
type alert, error, success Style of the notification. (Customizable with CSS)
modal false Adds modal layer when set to true.
speed 500 (ms) Speed of open and close animations.
timeout 5000 (ms), false Delay for closing event. Set false for sticky notification.
animateOpen {height: 'toggle'} A map of CSS properties.
animateClose {height: 'toggle'} A map of CSS properties.
easing swing Easing effect.
closable false Enables the close button when set to true.
closeOnSelfClick true Close the noty on self click when set to true.
closeOnSelfHover false Close the noty on self mouseover when set to true.
force false Adds notification to the beginning of queue when true.
buttons false, array An array of buttons.
onShow false, function onShow Callback (Details are in Callbacks section)
onClose false, function onClose Callback (Details are in Callbacks section)

Theme

noty is easily themable using CSS. But my English is not enough for the explain how. :(

Theme structre is in noty_theme_default.css file. You can copy-paste this file and change.

After that you can set your new theme with noty theme property. Like;

					
$('a.noty').click(function() {
	noty({
		text: 'noty - a jquery notification library!',
		theme: 'your_new_theme',
	});
});

Buttons

We can set button objects with an array like above; 
							
$('.ex1.confirm').click(function() {
	noty({
		text: note, 
		buttons: [
	    {type: 'button green', text: 'Ok', click: function($noty) {

					// this = button element
					// $noty = $noty element

					$noty.close();
					noty({force: true, text: 'You clicked "Ok" button', type: 'success'});
				}
	    },
	    {type: 'button pink', text: 'Cancel', click: function($noty) {
					$noty.close();
					noty({force: true, text: 'You clicked "Cancel" button', type: 'error'});
				}
	    }
	    ],
	  closable: false,
	  timeout: false
	});
	return false;
});
Available button properties are listed below.
Name Type Description
type string Button class
text string Button text
click function($noty) {} Callback function of button

Callbacks

Noty plugin have 2 callback option for now. We will add more callback asap.
Name Description
onShow Triggered when notification open animation finished.
onClose Triggered when notification close animation finished.

Noty API

Function Params Description
$.noty.clearQueue() none Clears the notification queue. (run)
$.noty.close() none Close the current notification. (run)
$.noty.closeAll() none Close all notifications. (run)
